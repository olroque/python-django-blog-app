The Blog is a full-stack web application developed using Python and the Django framework.


The site functions much like a social network. Users can create, edit, and delete posts, view posts by other users, edit profile info, upload profile pictures, and much more.


Feel free to explore the site. Register a new account, login, and create a post. Try adding a profile picture. If you forget your login info, click the Forgot Password link on the Login page. Click around and have fun!
